Introduction

formalia.js stores the content filled into a form, and recover this information when a user re-visits the page. This is the prevent that the end-user loses whatever she typed into the form when she is experiencing any one of these everyday emergencies:

  • The browser crashed.
  • Forgot that I was filling out a form in another tab, closed browser.
  • Accidentally clicked back-button.
  • Accidentally clicked a link.

formalia.js is a jQuery plugin still in its infancy. There is a bunch of features that should be implemented before it is production-ready, including:

  • Support ajax-loaded forms.
  • Simple user interface to save and load forms
    • For instance, when user revisits a page with a form, a message such as "Previously unsubmittet content in this form detected. Autofill form with content?"
    • A button for clearing form, with an undo function (or something).
